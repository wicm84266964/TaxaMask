# TaxaMask 2.3.1 内嵌 Ant-Code 1.3 与界面稳定性修复

日期：2026-07-12

版本号：`v2.3.1`

这是 `v2.3.0` 之后的小型维护版本。它把已经在 TaxaMask 中完成适配和人工验收的 Ant-Code 1.3 内嵌能力正式纳入稳定发布线，并修复认证、配置回退、窄屏导航和浅色/深色主题问题。

## 研究工作流影响

- “询问 Agent”和“启动智能体”恢复正常：TaxaMask 会先建立本地 Dashboard 认证会话，再完成状态检查、项目信任、历史会话读取和安全关闭。
- 长对话可以按页加载较早记录；研究者可见记录与模型恢复上下文分开归档，内部引导内容不会混入可见历史。
- 后台子智能体和后台终端在主回答结束后仍保留状态，重新连接活动会话时更容易判断任务是否仍在运行。
- 项目模型配置按字段覆盖环境默认值。项目未保存某个模型、网关字段或 API Key 时，可以继续使用启动环境中的对应值。
- 内嵌面板不再显示独立浏览器版的“会话 / 对话 / 文件”移动端底部导航，避免遮挡输入区。
- Ant-Code 1.3 新增状态已同时适配 TaxaMask 浅色和深色主题，包括长记录加载、连接状态、权限与完全访问确认、需求确认、队列、后台任务和模型配置。

## 安全与数据边界

- Dashboard API 使用端口隔离的 Cookie 和 CSRF 校验；修改状态的请求必须同时携带认证 Cookie 与 CSRF 请求头。
- 模型配置采用原子写入和并发保护，写入失败时保留原配置，降低配置文件半写入或互相覆盖的风险。
- 文件预览会拒绝通过符号链接或 Windows 目录联接跳出当前项目目录。
- 默认仍以本地项目为配置边界，不自动读取用户目录中的独立 Ant-Code 全局配置。
- TaxaMask 源码写入守卫、首问无旧对话时不压缩、空正文一次补救和单次工具结果 256 KiB 上限保持不变。
- 本版本不包含 API Key、本地 `.lab-agent/config.json`、项目数据库、模型权重、PDF/TIF 数据或运行产物。

## 内嵌版本范围

内嵌运行时版本为 `1.3.0-taxamask.1`，对齐 Ant-Code `v1.3.0` 并包含其标签后的 Windows 文件校验修复 `ca4e005`。

本次没有迁移独立版 EXE/发布流水线、Git 一等工具或大型 TUI 输入编辑器重构。独立浏览器 Dashboard 继续保留响应式移动端导航；只有 TaxaMask Qt 内嵌模式隐藏该导航。

## 验证与验收

- 内嵌运行时语法检查、Dashboard 构建资源一致性检查和网关兼容检查通过。
- Dashboard 浏览器交互测试 11/11 通过。
- Qt GUI smoke 96 个断言通过，覆盖真实 Cookie/CSRF 预检和浅色/深色主题契约。
- TIF Agent 上下文与 Agent 契约测试通过。
- 真实本地 Ant-Code 1.3 服务完成 bootstrap、status、trust、sessions 和 shutdown 认证闭环。
- 上游扩展测试中，TaxaMask 支持范围内的 Dashboard、运行时、服务、文件、会话和配置行为通过；只断言独立版用户全局配置语义的测试不属于 TaxaMask 内嵌支持范围。
- 研究者已完成主要交互项人工验收，暂未发现问题。该结论不表示 TaxaMask 的 2D、Blink、VLM、PDF、导出和全部 TIF 扩展路径均已逐项人工测试。

## 升级说明

从 `v2.3.0` 升级不需要迁移 TaxaMask 项目数据或数据库结构。首次启动内嵌 Agent Center 时，新的 Dashboard 会按 Ant-Code 1.3 认证方式建立本地会话；已有 Agent 历史会话继续由项目内 `.lab-agent/sessions/` 管理。
